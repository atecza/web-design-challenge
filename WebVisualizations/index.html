ABOUT

The WeatherPy project was undertaken through the University of Denver's Data Science Bootcamp. WeatherPy was created to evaluate the impact of latitude on weather conditions around the globe. Relying on data from the OpenWeatherMap API, WeatherPy randomly sampled over 600 cities on a single day (May 20th, 2020).

API calls for WeatherPy were conducted using python. Data was cleaned with the Pandas library and all plots were created using MatPlotLib.
